# Фотогалерея

Интерактивное веб-приложение для публикации и просмотра фотографий.

## Возможности

- Регистрация и авторизация пользователей
- Загрузка фотографий с указанием названия
- Удаление своих фотографий
- Лайки фотографиям других пользователей
- Редактирование профиля (имя, описание, аватар)
- Просмотр фото в полном размере

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, адаптивная верстка)
- JavaScript ES6+ (модули)
- Vite (сборка проекта)
- REST API

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск сервера разработки
npm start

# Сборка для продакшена
npm run build
```

## Валидация форм

- Имя: 2-40 символов, только буквы, пробелы и дефис
- Описание: 2-200 символов
- Название фото: 2-30 символов
- Ссылки: валидный URL

## API

Приложение работает с REST API сервера Яндекс.Практикума.
